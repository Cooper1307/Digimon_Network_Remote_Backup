# MTPE 汉化圣经：第二章 第一部分：落枭 (Falling Owl) v1.0

## Module A: 深度逻辑拆解

### [A.1] 英雄的缺位

**核心场景：** 樱从酒吧（喝一杯酒吧）醒来。枭的缺位突显了她的脆弱感，标志着她从被救助者的身份转变为必须独自在系统边缘求生的主角。
**语调主权：** 严格遵循“成熟”语调。将“痴淫美梦”处理为“delirious, feverish dream”，既保留了对救赎者潜意识的强烈吸引力，又避免了低级趣味。

---

## Module B: 词汇精析

### 1. **Decompilation (反编译/数字化解构)**

- **项目定义：** 一种不可逆的状态。由于身份验证硬件（体验勋章）被毁，人类灵魂/数据被数码世界的逻辑完全同化。
- **演化逻辑：**
  - [Human] -> [Badge Destruction] -> [Decompilation] -> [Bug/Data Dust]

### 2. **Drink a Cup Bar (喝一杯酒吧)**

- **内定：** 这是一个充满赛博朋克黑色电影风格的据点，是昆虫型数码兽的聚集地。
- **法则：** 中立地区，情报是这里的核心货币。

## Module C: 翻译原则（本章重点）

- **动词为核：** 在战斗和情感反应中使用更具侵略性的动词。
- **暗黑幻想润色：** 亚当（力神比多兽）的台词应带有愤世嫉俗、阅历丰富的气质。他的法语背景应在选词中体现出一种精英傲慢感。

---

## Module D: 设定审计与映射

- **亚当 (Adam):** 力神比多兽。身为“喝一杯酒吧”会长，他是个清高的巴黎中年男子形象。他与枭（Owl）相识多年，在中庭（Midgard）灰色地带极具影响力。
- **秩序兽 (Ordinemon):** 设定初登场。拉贵尔兽与座天使兽堕落形态的融合，象征着崩坏的秩序。
